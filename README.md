### Examples

```
If (True)
UDP GET SERVER LIST ($p1;$p2;$p3)
Else 
$wait:=1
$port:=19813
UDP GET SERVER LIST ($p1;$p2;$p3;$wait;$port)
End if 
```

By default the wait is ``1`` second and the port is ``19813`` i.e. ``0x4D65``.
